# Python Quiz Game
A simple quiz game I made when i was learning python.
A terminal quiz game made with Python.
#How to Run the Quiz Game
Make sure you have Python installed on your computer.
Download or clone this repository.
Open the project folder in your terminal.
Run the game with:
python main.py
Use the menu to:
Start the quiz
View your score
View the high score
Quit the game
#How to Play
Choose a difficulty level (Easy / Medium / Hard).
Each question has four options: A, B, C, D.
Type the letter of the correct answer.
Answer before the time limit runs out.
Your final score will be shown at the end.
Features:
- Randomized questions
- Difficulty levels
- Timer system
- High score saving
